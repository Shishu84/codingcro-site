# CodingCro Site

Welcome to **CodingCro**, a creative and technical hub built by [Shishu Kumar](https://shishu84.github.io/codingcro-site/).  
This site showcases projects, branding experiments, and interactive web development demos — blending **code, creativity, and communication**.

---

## 🚀 Features

- **Portfolio Showcase**: Modern, interactive portfolio highlighting web development and graphics projects.  
- **Creative Branding**: Unique logos, thumbnails, and channel branding experiments.  
- **Interactive Demos**: Web-based graphics (Canvas, SVG, Three.js, Python Turtle-inspired designs).  
- **Educational Content**: Beginner-friendly coding snippets, branding tips, and technical write-ups.  
- **Responsive Design**: Optimized for desktop and mobile viewing.

## 📖 How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/shishu84/codingcro-site.git
   ```
2. Navigate into the project folder:
   ```bash
   cd codingcro-site
   ```
3. Open `index.html` in your browser to explore the site locally.
